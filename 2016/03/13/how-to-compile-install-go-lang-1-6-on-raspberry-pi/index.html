在树莓派上编译安装go1.6

发表于 2016-03-13 | 分类于 go ， linux ， raspberry

在2月16号，golang团队宣布go 1.6正式发布，在系统要求栏其实可以看到golang在linux平台是支持arm架构的

Operating system Architectures Notes
FreeBSD 9.3 or later amd64, 386 Debian GNU/kFreeBSD not supported
Linux 2.6.23 or later with glibc amd64, 386, arm, arm64,s390x, ppc64le CentOS/RHEL 5.x not supported.Install from source for other libc.
macOS 10.8 or later amd64 use the clang or gcc† that comes with Xcode‡ for cgo support
Windows XP SP2 or later amd64, 386 use MinGW gcc†. No need for cygwin or msys.

可以看出树莓派2是完全满足要求的

准备编译器

go的工具同样是使用go语言写成的，所以这成了一个鸡生蛋蛋生鸡的问题，我在这使用了Dave Cheney制作的go 1.4 编译器，官网同样可以找到，我在这就不去试了

curl http://dave.cheney.net/paste/go-linux-arm-bootstrap-c788a8e.tbz | tar xj
