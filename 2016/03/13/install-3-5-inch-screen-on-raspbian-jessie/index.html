树莓派raspbian-jessie安装3.5英寸lcd屏幕

发表于 2016-03-13 | 分类于 linux ， raspberry

最近在某宝购入了一个树莓派的屏幕
差不多就是这样子

通过GPIO连接到树莓派，一切看起来都那么完美，但是插入后却一直白屏，客服甩过来一个链接大致就是下一个驱动运行，reboot就行，然而并没有，客服也没有解决方法，我也懒得退货，因为我深信屏幕是没问题的是我的姿势有问题

多谢google 花了一晚上的时间，总算解决了这个问题

首先关于3.5 inch lcd的关键字将我带入waveshare这个网站，这个网站就是卖adruino和树莓派配件的，在一篇介绍他家3.5英寸屏幕的wiki)上，我似乎找到了解决方案，但是仔细一看，这不就是客服给我的解决方案吗，连驱动目录内的文件都是一样的，我有一个猜想，某宝的屏幕多半是仿造的，驱动也是兼容的

但是为啥不管用呢，有没可能是系统的问题，我的系统是15年2月烧的，来到树莓派官网发现最新的raspbian已经是

基于debian-jessie的raspbian-jessie了，抱着死马当活马医的心态，我重新烧制了最新的系统，然后运行了之前的驱动，然并卵，正当我绝望的时候，突然我在wiki)发现了一个不起眼的链接

下载驱动

wget http://www.waveshare.com/w/upload/9/9d/LCD-show-151020.tar.gz

解压并运行驱动

配置图形化启动

因为我不使用图形化界面，所以并没有配置boot选项，如果需要以图形界面进入请进行如下配置
sudo raspi-config

依次选择

Enable Boot to Desktop/Scratch -> Desktop Log in as user 'pi' at the graphical desktop
然后重启，你就能看到大大的树莓LOGO了
