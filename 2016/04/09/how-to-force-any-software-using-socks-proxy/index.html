在windows下强制任何程序使用socks代理

发表于 2016-04-09 | 分类于 科学上网

我在之前的博文有讲过在linux上如何使用tsocks让不支持socks代理的程序强制使用socks代理，使用tsocks将socks代理转为http代理

今天讲一讲在windows环境下如何让程序强制走socks代理

工具

主要使用的是国人作者Taro写的www.sockscap64.com

最新版本更是加入了对shadowsocks的支持，项目本身也是在sourceforge上开源的，不用担心后门什么的，因为是图形界面操作，使用起来非常方便

运行&配置

推荐在sourceforge项目地址下载 (需梯子)，下载最新免安装版SocksCap64-Portable-3.x.zip 解压即可

解压后运行sockscap64.exe 即可看到图形界面

我们可以添加本地代理，也可以直接添加远程Shadowsocks代理，通过手动或者扫描二维码的形式添加非常方便

配置好代理后，点击程序加入我们想使用socks代理的程序即可运行，就是这么简单

# shadowsocks # socks5 # sockscap
使用tsocks将socks代理转为http代理
