Mac下如何配置privoxy完成go get代理

发表于 2016-04-11 | 分类于 go ， MacOS ， 科学上网

我在MacOS下解决终端代理问题讲解了如何使用proxychains完成终端代理

使用tsocks将socks代理转为http代理，也讲解了如何使用socks代理，但是Mac下tsocks好像并没有很好的解决方案，brew官方好像也移除了tsocks，我们就来使用privoxy作为替代。

关于privoxy

官方对于privoxy的解释是一款无缓存的web代理工具，主要用于过滤垃圾广告。

下载privoxy

项目主页在sourceforge上，找对应你系统的版本下载

安装privoxy

直接双击下载好的文件，一步步确认即可完成privoxy的安装。

配置privoxy

打开终端，输入以下命令

cd /Applications/Privoxy

监听8118端口

echo 'listen-address 0.0.0.0:8118' >> /usr/local/etc/privoxy/config

设置转发socks5服务器

echo 'forward-socks5 / localhost:1080 .' >> /usr/local/etc/privoxy/config
安装好后privoxy已经是启动的了
sudo ./startPrivoxy.sh
