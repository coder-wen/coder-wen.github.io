发表于 2016-04-12 | 分类于 Code Stuff

markdown是什么

根据wiki百科，Markdown 是一种轻量级标记语言，创始人为约翰·格鲁伯（John Gruber）。它允许人们“使用易读易写的纯文本格式编写文档，然后转换成有效的XHTML(或者HTML)文档”。这种语言吸收了很多在电子邮件中已有的纯文本标记的特性
2016年，markdown可以说无处不见，github的README都是.md结尾的，不少开源书籍也开始使用markdown进行编辑，还不学学markdown你就out了

准备工具

  • 首推跨 Linux/MacOS/Windows 三平台Atom，自带markdown插件，编辑以及预览，接下来的课程我都使用atom来渲染演示

  • MacOS

    • MacDown 推荐免费开源的MacDown
    • Mou 国人开发，口碑非常不错
    • typora 免费轻量级的markdown编辑器同时支持MacOS、Windows

  • Windows

    • typora 免费轻量级的markdown编辑器同时支持MacOS、Windows

  • Linux

    • Atom

列表

markdown也支持列表，无序列表和有序列表使用起来都非常简单

值得注意，这里*号换成+或者-效果是一样的

无序列表支持子列表，在父列表的下面那一行使用两个空格来表明子列表，在atom里tab默认是两个空格，所以直接tab即可

段落

之前我们使用了空行使markdown实现换行，这叫做hard breaking，太暴力了，我们还是来个soft breaking
不少文艺气息的码农说我要写首诗都要多敲那么多空行，soft breaking非常简单只需在句尾加上两个空格

不要忘了句尾的空格哦

对于短小的字块高亮，我们使用反引号将其包围。

对于长文本来说双反引号就略显蛋疼了，我们之前说过识别换行就两种办法，softbreaking or hardbreaking，几十上百行谁和你breaking，蛋疼，这个时候就需要三反引号包围文本
