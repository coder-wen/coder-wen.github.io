为kitematic设置代理加速docker pull

发表于 2016-04-30 | 分类于 Docker ， MacOS ， 科学上网

kitematic是docker推出的GUI工具，使操作docker的方式变得更简单直观，但是直连docker hub的速度也是够呛，今天就来说说如何加速kitematic从docker hub拉取镜像

2016.08.02更新

Docker官方已经推出了docker for mac，代理设置方便了很多，直接进到docker的preferences，找到advanced，填上代理服务器，记得是物理ip，不要填127.0.0.1

重启就可以愉快的下载镜像了
