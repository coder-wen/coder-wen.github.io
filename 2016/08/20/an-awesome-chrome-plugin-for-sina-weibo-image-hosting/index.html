好用的Chrome插件新浪微博图床插件

发表于 2016-08-19 | 分类于 工具

今天在某论坛发帖要插入图片，虽然自己有七牛的CDN，但是挂到第三方网站也不太好，一是流量扛不住，还有可能遭受报复打击，寻思着还是得用个免费的图床，微博是个不错的选择。

手动搞也不是个好办法，程序员就是得偷懒，在Chrome商店里还真发现了个好东西，微博图床，用了一下真是神器也非常感谢作者Sam su的无私奉献，插件本身也是开源的，地址

下面这张图就是用微博图床插件上传的

  • 使用方法

    • 在Chrome上登录微博(不登陆肯定没权限，想都别想)

    • 点击图标，拖拽上传，自动给出四种链接

      *   单纯的URL
      • HTML
      • MarkDown
      • UBB

  • Feature

    • 可以查看上传历史
    • 可以启用HTTPS ( 简直不要太爽 :)
    • 可以删除上传
    • 可以选择尺寸
