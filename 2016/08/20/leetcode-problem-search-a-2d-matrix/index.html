题目

这次的题目是Search a 2D Matrix

难度：middle

思路

其实二维不二维都没有太大关系了，我们就可以把他看作一个一维有序数组，寻找某个值存不存在。有序数组里面搞查询，肯定是二分查找最快啊。O(logN)的时间复杂度

总的来说还是比较简单的。

代码

二分查找我习惯使用了递归实现，你想用循环当然可以。

算法复杂度O(LogN)
